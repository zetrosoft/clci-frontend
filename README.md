# clci-frontend

Production build for the CLCI frontend, published as a static site artifact.

## Overview

This repository contains the compiled output of the CLCI frontend application. It is intended to be deployed as a static web app or served from a static file host.

## Contents

- `dist/` - production build output generated by Vite
- `LICENSE` - MIT license for this repository

## Usage

You can serve the app with any static file server. For example:

```bash
npx serve dist
```

Or deploy the `dist/` directory to your hosting platform of choice.

## Build Source

The source code for this frontend lives in the main `clci` repository. To update this repository:

1. Make changes in the frontend source.
2. Run the frontend build.
3. Replace the contents of `dist/` with the fresh build output.
4. Commit and push the updated artifact here.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
